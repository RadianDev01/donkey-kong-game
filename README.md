# 🦍 Donkey Kong - Arcade Game

A simple Donkey Kong-inspired arcade game built with vanilla HTML5 Canvas and JavaScript. No dependencies, no build tools — just open and play!

## 🎮 Play Now

**[Play the game →](https://radiandev01.github.io/donkey-kong-game/)**

## Controls

| Key | Action |
|-----|--------|
| ← → | Move left/right |
| ↑ ↓ | Climb ladders |
| Space | Jump |

Mobile touch controls are also supported.

## Features

- Classic DK-style platform gameplay
- Barrel dodging with physics
- Ladder climbing
- Multiple levels with increasing difficulty
- Score tracking with local high score
- Retro pixel-art style graphics
- Mobile responsive with touch controls

## How to Play

1. Dodge the barrels thrown by Donkey Kong
2. Climb ladders to reach higher platforms
3. Jump over barrels for bonus points (+100)
4. Reach the princess at the top to win!
5. Each level gets faster and harder

## Tech

- Pure HTML5 Canvas
- Vanilla JavaScript (no frameworks)
- CSS3 for UI overlays
- Zero dependencies

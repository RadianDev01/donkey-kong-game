# 🦍 Donkey Kong - Arcade Game

A retro-style Donkey Kong arcade game built with HTML5 Canvas and vanilla JavaScript.

## 🎮 Play Now

[Play the game](http://172.104.189.52/donkey-kong/)

## ✨ Features

- **Classic Gameplay** — Climb ladders, dodge barrels, rescue the princess!
- **🔨 Hammer Power-up** — Grab the hammer to smash barrels for bonus points
- **🔊 Retro Sound FX** — Synthesized 8-bit sounds using Web Audio API (no external files needed)
- **📱 Mobile Support** — Touch controls with D-pad and jump button
- **⏸ Pause** — Press P or tap the pause button
- **🎵 Mute Toggle** — Press M or tap the sound button
- **💥 Particle Effects** — Explosions when barrels are destroyed
- **📊 Combo System** — Chain barrel jumps/smashes for bonus points
- **🖥 Screen Shake** — Impact feedback on player death
- **⏱ DeltaTime** — Consistent game speed regardless of frame rate
- **🏆 High Scores** — Saved locally via localStorage
- **📈 Progressive Difficulty** — Barrels get faster each level

## 🎹 Controls

| Key | Action |
|-----|--------|
| ← → | Move left/right |
| ↑ ↓ | Climb ladders |
| Space | Jump / Start |
| P | Pause |
| M | Mute/Unmute |

## 🛠 Tech Stack

- HTML5 Canvas
- Vanilla JavaScript (no dependencies)
- Web Audio API for sound effects
- CSS3 for UI overlays

## 🚀 Deploy

Just serve `index.html` — no build step required!

```bash
# Quick local server
npx serve .
```
